### Laravel + React = SSR

##### How to install:

requirements:

-   Composer https://getcomposer.org/doc/00-intro.md
-   Laravel https://laravel.com/docs/5.8/installation

1. Clone repository
2. `$ composer install`
3. `$ npm install`

##### How to run project:

1. `npm run dev` || `npm run watch`
2. `$ php artisan serve`
